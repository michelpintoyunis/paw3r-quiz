# PAW3R Quiz

Una aplicación de Quiz interactiva desarrollada con React y Vite para la marca PAW3R.

## Descripción

Esta aplicación es un cuestionario diseñado para poner a prueba el conocimiento sobre la marca PAW3R. Incluye efectos de sonido, diseño responsivo y una interfaz animada.

## Ejecutar localmente

1. Instalar dependencias:

   ```bash
   npm install
   ```

2. Ejecutar la aplicación:
   ```bash
   npm run dev
   ```

## Tecnologías

- React
- Vite
- TypeScript
- Tailwind CSS
